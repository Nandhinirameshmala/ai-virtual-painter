# 🎨 AI Virtual Painter 🤖✋

A real-time AI-powered virtual painting application using **Python, OpenCV, and MediaPipe Hand Tracking**.  
Draw, erase, and control colors in the air using only your fingers — no mouse or touchscreen required.

---

## ✨ Overview

This project uses computer vision to detect hand landmarks and converts finger movements into drawing actions on a virtual canvas.

---

## 🚀 Features

- ✋ Real-time hand tracking using MediaPipe
- ☝️ Draw using index finger
- ✌️ Switch to selection mode using two fingers
- 🎨 4 bright colors (Blue, Green, Red, Yellow)
- 🧽 Eraser mode (gesture-based)
- 🗑️ Clear canvas option
- 🖥️ Smooth 640×480 interface
- ⚡ Real-time performance

---

## 🧠 How It Works

Webcam captures hand → MediaPipe detects 21 landmarks → Finger gestures are analyzed → Actions are performed on a virtual canvas.

---

## 🎮 Controls

- ☝️ One Finger → Draw
- ✌️ Two Fingers → Select color/tools
- Move finger on color box → Change color
- Move on 🧽 ERASER → Erase drawing
- Move on 🗑️ CLEAR → Reset canvas
